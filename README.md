# bobs-repository — Install Guide Web App

A simple, static **install-guide web app** that walks users through setting up the project step-by-step.

## Features

- Step-by-step installation checklist with progress tracking
- One-click copy for every shell command
- Progress persisted in `localStorage` across page refreshes
- Fully responsive — works on desktop and mobile
- No build step required — open `index.html` directly in any browser

## Usage

Open `index.html` in your browser, or serve it with any static file server:

```bash
npx serve .
```

Then visit <http://localhost:3000>.
